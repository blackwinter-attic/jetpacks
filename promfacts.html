About

This is a demo Jetpack feature that displays prometheus Facts. (View source)

Install

Get Jetpack and click "Install..." on the bar that appears on the top of this page when Jetpack is installed.

Configuration

Language

Got to about:config and add a new String preference jetpacks.promfacts.lang with a value corresponding to the L parameter for the prometheus Homepage. (Currently supported are 1 for English, and 0 for German.)
